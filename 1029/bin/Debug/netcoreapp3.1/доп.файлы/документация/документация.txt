Инструкция:

Система
для открытия файла или папки укажите путь и используйте '/' (пример: C:\Users\user\Desktop\file.txt/)
для отключения ПК используйте команду - power off\
для добавления файлов в список используйте команду 'add' (пример: add filename
								                                   add fileroad\)
для открытия файла введите псевданим, который вы указали при добавлении файла (пример: filename)

Интернет
для открытия ссылку введите ее и испольщуйте '\' (пример: https://r0d17n.github.io/1029site/index.html\)
для того, чтобы добавить браузерную ссылку в список используйте команду add (пример: add sitename
										                                               add link\)
для того, чтобы добавить пользователя для отправки сообщений в телеграм используйте команды name и id (пример: name username
                                                                                                                                                                                              id userid)
для того, чтобы отправить сообщение пользователю телеграм используйье команду M или m в первой строке для ввода сообщения
и псевдоним пользователя (пример: M message
                                                           username)
для того, чтобы открыть ссылку из списка введите псевданим, который вы указали при добавлении файла (пример: sitename)
для того, чтобы открыть ссылку не имеющую псевдоним, просто введите ее в терминал (пример: https://link.com\)
для того, чтобы выполнять поиск в браузере используйте команду 's' или 'S' (пример: s text\)
для того, чтобы выполнять поиск на сайте youtube используйте команду 'y' или 'Y' (пример: y videoName)
для того, чтобы выполнять поиск на сайте kinopoisk используйте команду 'kp' или 'KP' (пример: kp filmName)
для того, чтобы выполнять поиск на сайте aliexpress используйте команду 'ali' (пример: ali productname)

Вспомогательные
для того, чтобы удалить временный текстовый файл используйте команду 'delete'
для того, чтобы отчистить списки путей и названий файлов/ссылок используйте команду 'clear'
для того, чтобы отчистить терминал используйте команду 'cls'
для того, чтобы отрыть/редактировать исходный код терминала используйте команду 'mod_code'
для того, чтобы открыть документацию помощника используйте команду 'doc'
для запуска секундомера используйте команду SW или sw
для запуска таймера используйте команду mtimer (пример: 60 mtimer)
